## 文件传输

- 使用 rz 传输文件
- 使用 Cmd + Shift + P 传输文件(直接拖船或put)

<!--more-->

## 命令

- Ctrl + l 清屏
- Ctrl + a 跳转到行首
- Ctrl + e 跳转到行末
- Cmd + Enter 全屏
- Cmd + K 快速创建新Session
- Option + Cmd + m 切换到SessionManager
- Option + Cmd + g 切换到CommandWindow （可以给所有Session发送命令）
